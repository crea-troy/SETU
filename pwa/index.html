How to export from ChatGPT:
1. chatgpt.com → Profile → Settings
2. Data Controls → Export Data
3. Wait for email → Download ZIP
4. Extract ZIP → find conversations.json → upload here
How to export from Claude:
1. claude.ai → Profile → Settings
2. Privacy → Export conversations
3. Download the JSON file → upload here
Any other AI tool:
If your AI tool exports chat history as JSON — upload it here. SETU will try to parse it and extract what's useful.
